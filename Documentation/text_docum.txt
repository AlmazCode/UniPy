Текстовый объект в UniPy представляет собой сущность, которая может быть помещена на игровую сцену и взаимодействовать с другими объектами.

Список параметров, доступных для редактирования в текстовом объекте:

    x: координата по оси X, задающая позицию текстового объекта на игровой сцене.
    y: координата по оси Y, задающая позицию текстового объекта на игровой сцене.
    text: текст, который будет отображаться на объекте.
    fontSize: размер шрифта текста.
    font: шрифт, используемый для отображения текста.
    color: цвет текста.
    transparent: прозрачность текста.
    angle: угол, под которым объект отображается на экране
    textCentering: центрирование текста по определенным сторонам: left - с лева, centerx - по центру, right - справа.
    render: если значение этого параметра равно False, то объект не будет отображаться на экране.
    flipX: если значение этого параметра равно True, то объект будет отражен по горизонтальной оси.
    flipY: если значение этого параметра равно True, то объект будет отражен по вертикальной оси.
    cx: центрирование объекта по оси X по определенным сторонам: right - центрирование с правой стороны экрана, left - с левой, centerx - с центра экрана,objX(ObjectName) - с позиции объетка по оси X.
    
    cy: центрирование объекта по оси Y по определенным сторонам: top - центрирование с верхней стороны экрана, bottom - с нижней, centery - с центра экрана, objY(ObjectName) - с позиции объетка по оси Y.
    
Еще, обьект будет центрироваться только при старте программы! И чтобы еще раз отцентрировать объект по его параметрам, нужно будет в коде вызывать функцию, их две - setPos: right, left, top, bottom, centerx, centery, setPosObject: objX, objY.

Один из важных функциональных возможностей текстового объекта в данном движке - поддержка многострочного текста. Для того, чтобы разбить текст на несколько строк, можно использовать специальный символ переноса строки "\n". В эдиторе "\n", в коде "\\n"

Например, если мы хотим отобразить следующий текст на двух строках:
"Hello, world!
This is a multiline text."

Мы можем ввести его в свойство text объекта следующим образом:
"Hello, world!\nThis is a multiline text."

Таким образом, наш текст будет отображаться на двух строках, и после слова "world" будет добавлен символ переноса строки, который создаст новую строку для следующей части текста.

Использование многострочного текста может быть очень полезным при создании игровых объектов, которые содержат длинные описания, диалоги, инструкции или другую информацию. Работа с таким текстом становится более удобной и простой благодаря возможности переноса текста на новую строку.

Знание этих параметров позволяет эффективно управлять текстовым объектом в движке и настраивать его внешний вид и поведение в игре. Редактирование параметров можно производить как вручную, так и с помощью возможностей эдитора, что позволяет ускорить процесс создания игровых объектов и разработку игры в целом.
