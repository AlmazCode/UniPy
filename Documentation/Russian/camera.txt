В UniPy камера создает небольшой "мини-объект", который отслеживает движение выбранного объекта.
Работа с камерой в этом движке находит баланс между простотой и сложностью.
Чтобы назначить объект для отслеживания камерой, используйте функцию `SetCamera`, вот так:

```python
import UniPy as up

up.SetCamera("player")
```

В этом коде мы импортируем функцию и используем её. Не забудьте указать имя объекта из иерархии редактора!

Более того, вы можете настроить горизонтальное и вертикальное положение камеры. Это приведет к тому, что камера будет сосредоточена на объекте с добавленными полями с обеих сторон на основе значений x и y. Например:

```python
import UniPy as up

up.SetCamera("player")
up.Camera.y = -100
```

Вышеуказанный код сдвигает камеру вверх на 100 пикселей.
