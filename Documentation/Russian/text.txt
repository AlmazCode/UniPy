Объект текста в UniPy служит как элемент, размещенный внутри игровой сцены, способный взаимодействовать с другими объектами.

Параметры, доступные для редактирования в объекте текста, включают:

- `x`: X-координата, определяющая позицию объекта текста на игровой сцене.
- `y`: Y-координата, определяющая позицию объекта текста на игровой сцене.
- `text`: Текст, который будет отображаться на объекте.
- `fontSize`: Размер шрифта текста.
- `font`: Шрифт, используемый для отображения текста.
- `color`: Цвет текста.
- `transparent`: Прозрачность текста.
- `angle`: Угол, под которым объект отображается на экране.
- `textCentering`: Центрирование текста по конкретным сторонам: слева, по центру по оси X, справа.
- `render`: Если False, объект не будет отображаться на экране.
- `flipX`: Если True, объект будет отражен горизонтально.
- `flipY`: Если True, объект будет отражен вертикально.
- `richText`: Если True, текст будет поддерживать многоцветную систему.
- `cx`: Центрирование объекта по оси X по конкретным сторонам.
- `cy`: Центрирование объекта по оси Y по конкретным сторонам.
- `script`: Хранит скрипты для управления объектом.
- `layer`: Слой для рисования объекта.
- `tag`: Тег объекта для фильтрации или поиска.
- `useCamera`: Если True, объект будет отображаться из камеры.
- `smooth`: Если True, текст будет отображаться с сглаживанием.

Более того, объект будет центрирован только при запуске. Чтобы перецентрировать его на основе параметров, в коде должна быть вызвана функция. Доступно две функции: `setPos` (right, left, top, bottom, centerx, centery) и `setPosObject` (objX, objY).

Поддержка многострочного текста в движке заслуживает внимания. Чтобы разделить текст на несколько строк, используйте символ новой строки "\n". Например, чтобы отобразить текст "Привет, мир!" и "Это многострочный текст." на отдельных строках, введите это как "Привет, мир!\nЭто многострочный текст."

Этот движок также поддерживает многоцветную систему с использованием тегов. Начните тег с символа доллара "$", за которым следует цвет текста в формате RGB в круглых скобках. Например:
```
$(255, 0, 0)Привет, $(0, 255, 255)мир!
```
В этом примере "Привет" будет красным, а "мир!" - бирюзовым.

Понимание этих параметров позволяет эффективно управлять объектами текста, настраивать их внешний вид и поведение в игре. Параметры можно редактировать вручную или через редактор, упрощая процесс создания игровых объектов и общего развития игры.
